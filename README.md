# Swag Labs – Playwright UI Tests

Automatisierte UI-Tests für die Swag Labs Demo-Applikation, umgesetzt mit Playwright (Java) und JUnit 5.  
Das Projekt dient als Portfolio-Beispiel für strukturierte UI-Testautomatisierung mit Page Objects.

## Tech Stack
- Java
- Playwright
- JUnit 5
- Maven

## Run
```bash
mvn test
